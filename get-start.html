Installation

Memasang FramaDza sebagai alat bantu anda untuk membuat website artinya anda sudah menyetujui Privacy Policy. Baca terlebih dahulu jika belum. Berikut detail update terbaru dari FramaDza saat ini.

  • Name : FramaDza
  • Version : Alpha-0.0.1
  • Framework : CSS
  • Latest Update : 19 April, 2023
  • Author : Moh. Nurjaman


Tidak ada Ketentuan Minimum Perangkat Keras untuk menggunakan FramaDza, yang anda bisa lakukan adalah pasang saja sesuai dengan metode yang sudah disedikan.
Anda bisa gunakan Offline Installation atau Online Installation. Anda juga bisa ikuti cara Components Installation untuk memasang hanya komponen tertentu saja yang anda butuhkan untuk dipasang. Gunakan menu dikiri anda (Mode Desktop) atau diatas anda (Mode Mobile) untuk membuka Tutorial.

Offline Installation

Instalasi menggunakan Mode Offline artinya anda faham bahwa paket yang akan diinstall bersifat sementara dan tidak LTS (Long Time Service). Anda hanya bisa Update ke Versi terbaru dengan mengunduh paket baru lalu mengganti paket lama. Berikut cara pemasangan FramaDza Menggunakan Mode Offline

FramaDza Alpha v.0.0.1

  1. Pertama, unduh terlebih dahulu paket .ZIP FramaDza versi terbaru melalui link diatas.
  2. Ekstrak paket ke folder tempat anda akan membuat website.
    Contoh : C:/XAMPP/htdoc/my-website/Disini
    catatan : Hasil Ekstrak akan berupa Folder dengan nama fdz.
  3. Selanjutnya buat file.html anda. Atau jika sudah ada abaikan langkah ini
  4. Salin kode berikut :
    <link rel="stylesheet" href="fdz/css/main_framadza.css">
    dan Panggil/Referensikan FramaDza pada <head> index.html anda.
    <!DOCTYPE html>
    <html>
     <head>// Antara disini
      <meta charset="utf-8">
      <meta name="viewport" content="width=device-width, initial-scale=1">
      <link rel="stylesheet" href="fdz/css/main_framadza.css">
       <title>Website Anda</title>
     </head>// dan disini
    <body>
    </body>
    </html>
  5. Selesai. Anda tinggal panggil Class / Components yang anda perlukan pada class="". Baca Documentation untuk mulai menggunakan FramaDza.

Online Installation

Mode ini lebih praktis dan LTS (Long Term Service). Ketika ada Update Paket FramaDza terbaru, anda tidak perlu menginstall ulang FramaDza. Anda hanya perlu membuka detail update saja jika tertarik. Namun Mode ini tidak akan berfungsi jika anda tidak terkoneksi dengan internet. Berikut cara instalasinya

  1. Salin kode berikut :
    <link rel="stylesheet" href="https: //cdn.frama.dza.my.id/fdz/css/main_framadza.css">
  2. lalu salin diantara tag <head> pada file.html anda
    <!DOCTYPE html>
    <html>
     <head>// Antara disini
      <meta charset="utf-8">
      <meta name="viewport" content="width=device-width, initial-scale=1">
      <link rel="stylesheet" href=" https://cdn.frama.dza.my.id/fdz/css/main_framadza.css">
       <title>Website Anda</title>
     </head>// dan disini
    <body>
    </body>
    </html>
  3. Selesai. Anda tinggal panggil Class / Components yang anda perlukan pada class="". Baca Documentation untuk mulai menggunakan FramaDza.

Component Installation

Cara memasang Components website menggunakan FramaDza cukup mudah, ikuti cara berikut

  1. Pertama pasang terlebih dahulu FramaDza menggunakan Offline Installation atau Online Installation diatas.
  2. Buka Documentation lalu buka menu Components
  3. Cari komponen yang anda perlukan
  4. Salin kode komponennya dimana anda ingin menaruh komponennya, dan Selesai.
