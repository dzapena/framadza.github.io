FramaDza

FramaDza adalah sebuah Framework yang didistribusikan untuk penggunaan personal dan bukan untuk bisnis atau sejenisnya. FramaDza adalah Framework CSS yang berfokus pada penyederhanaan utilitas, fungsi dan kegunaan yang biasa digunakan untuk membuat sebuah website. Framework ini bisa digunakan oleh siapapun secara gratis dengan syarat tertentu sebagai berikut.

Privacy Policy

  1. Tidak untuk digunakan sebagai kejahatan, Membuat website Penipuan, Malware, bshdhs atau sejenisnya.
  2. Jangan edit kode dalam framework lalu mengaku bahwa ini adalah buatan seseorang.
  3. Tempelkan credits FramaDza pada footer website yang dibuat oleh website ini atau bisa di tempelkan pada menu about juga.
  4. Ditujukan untuk kegiatan edukasi/pembelajaran.
  5. Masih terus dalam pengembangan, karena ini adalah project eksperimental pribadi author.

Komunitas dan Q&A

Sejauh ini belum ada komunitas yang berkembang, tapi author menyiapkan beberapa Q&A terkait pertanyaan yang mungkin akan ditanyakan. Berikut record pertanyaan yang sudah terkumpul beserta jawabannya

  1. Kenapa FramaDza tidak menggunakan Javascript ? Lihat Jawaban

    • Cukup sederhana, author hanya belum menguasainya. Juga author menginkan framework yang menag benar benar hanya berdasakan CSS.

  2. Apakah FramaDza sudah mobile Friendly dan Responsive ? Lihat Jawaban

    • Tentu saja, bahkan anda bisa mengatur sendiri mau berapa pixel sampai website berubah jadi mode mobile. Atau lebih lanjutany lagi anda bisa membuat setiap mode atau pixel layar memiliki tampilan yang berbeda untuk website anda.

  3. Apa bedanya sama Bootsrap atau Tailwind ? Lihat Jawaban

    • Sejujurnya ini tidak ada bedanya dengan kedua Framework CSS tersebut, tapi authoe hanya ingin membuat FramaDza sebagai Tujuan pembejalaran dan juga pengembangan diri. Mungkin FramaDza akan memiliki banyak kelemahan dibanding kedua Framework terkenal tersebut. Tapi author berusaha sebaik mungkin membuat FramaDza menjadi FrameWork yang akan memiliki keunikan dan kelebihannya sendiri nanti
