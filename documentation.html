Border

Border adalah garis yang mengelilingi suatu atribut, border biasa digunakan juga untuk memberikan garis objek atau memberikan bayangan pada objek.

Border / Garis

  • Border Non-Color

No Class Fungsi
1 b-0 Memberikan Border 0 pixel
2 b-1 Memberikan Border 1 pixel
n b-n Memberikan Border N pixel
20 b-20 Memberikan Border 20 pixel

Tabel 1. Border

  • Border Color

Tambahkan huruf c sebelum kode pixel untuk memberikan warna Hitam pada border.

Contoh : jika anda ingin memasang border dengan ukuran 4 pixel dengan warna hitam
<class="b-c4">

Gunakan label berikut untuk warna lain :

Dalam versi ini warna lain belum tersedia

One Line Border / Garis Satu Jalur

  • One Line Border Non-Color

No Class Fungsi
1 b-bb-1 Memberikan Border garis bawah 1 Pixel
2 b-bb-2 Memberikan Border garis bawah 2 Pixel
n b-bb-n Memberikan Border garis bawah n Pixel
20 b-bb-20 Memberikan Border garis bawah 20 Pixel

Tabel 2. One Line Border

Ada 4 jenis garis saat membuat sebuah Border, yaitu Top (Garis Atas), Bottom (Garis Bawah), Left (Garis Kiri), dan Right (Garis Kanan)

Semua nya diwakili dengan label berikut :

  • bt untuk Border Top
  • bb untuk Border Bottom
  • bl untuk Border Left
  • br untuk Border Right

Untuk dapat menggunakan One Line Border anda cukup menganti kode bb pada tabel diatas menggunakan label pada list diatas

  • One Line Border Color

Tambahkan huruf c sebelum kode pixel untuk memberikan warna Hitam pada border.

Contoh : jika anda ingin memasang border bottom dengan ukuran 4 pixel dengan warna hitam
<class="b-bb-c4">

Gunakan label berikut untuk warna lain :

Dalam versi ini warna lain belum tersedia

Border Radius / Garis Radius

Garis yang menghasilkan lengkungan di setiap siku siku atribut. Anda dapat gunakan persentase atau pixel(px) untuk membuat Border Radius

  • Border Radius Non-Color

No Class Fungsi
1 br-p10 Memberikan Border Radius Sebesar 10%
2 br-p20 Memberikan Border Radius Sebesar 20%
n br-pn Memberikan Border Radius Sebesar n%
10 br-p100 Memberikan Border Radius Sebesar 100%

Tabel 3. Border Radius Menggunakan Persentase

No Class Fungsi
1 br-5 Memberikan Border Radius Sebesar 5 Pixel
2 br-6 Memberikan Border Radius Sebesar 6 Pixel
3 br-6 Memberikan Border Radius Sebesar 7 Pixel
n br-n Memberikan Border Radius Sebesar n Pixel
100 br-100 Memberikan Border Radius Sebesar 100 Pixel

Tabel 4. Border Radius Menggunakan Pixel

  • Border Radius Color

Tambahkan huruf c sebelum kode pixel untuk memberikan warna Hitam pada border.

Contoh : jika anda ingin memasang border radius dengan ukuran 5 pixel dengan warna hitam
<class="br-c5">

Penting : Border Radius yang menggunakan persentase tidak bisa menggunakan Warna

Gunakan label berikut untuk warna lain :

Dalam versi ini warna lain belum tersedia

Box Shadow / Bayangan Kotak

Menghasilkan bayangan pada Atribut tertentu yang dipasang kode ini

No Class Fungsi
1 bs-1 Memberikan Bayangan pada kotak dengan Transparasi 10%
2 bs-2 Memberikan Bayangan pada kotak dengan Transparasi 20%
n bs-nn Memberikan Bayangan pada kotak dengan Transparasi n%
10 bs-10 Memberikan Bayangan pada kotak dengan Transparasi 100%

Margin and Padding

Margin

Margin adalah jarak antara atribut dengan atribut lain.

  • Margin All Side / Margin Semua Sisi

No Class Fungsi
@ mg-a Memberikan Margin Auto
@ mg-0 Memberikan Margin 0 pixel
1 mg-1 Memberikan Margin 1 pixel
2 mg-2 Memberikan Margin 2 pixel
3 mg-3 Memberikan Margin 3 pixel
4 mg-4 Memberikan Margin 4 pixel
5 mg-5 Memberikan Margin 5 pixel
n mg-n Memberikan Margin N pixel
50 mg-50 Memberikan Margin 50 pixel

Tabel 1. Margin All Side

  • Margin One Side / Margin Satu Sisi

Sama seperti border, Margin juga punya 4 sisi yang bisa kita atur ukuran Marginnya masing masing yaitu Margin Bottom (Bawah), Margin Top (Atas), Margin Left (kiri) dan Margin Right (kanan).

No Class Fungsi
@ mgt-a Memberikan Margin Top Auto
@ mgt-0 Memberikan Margin Top 0 pixel
1 mgt-1 Memberikan Margin Top 1 pixel
2 mgt-2 Memberikan Margin Top 2 pixel
3 mgt-3 Memberikan Margin Top 3 pixel
4 mgt-4 Memberikan Margin Top 4 pixel
5 mgt-5 Memberikan Margin Top 5 pixel
n mgt-n Memberikan Margin Top N pixel
50 mgt-50 Memberikan Margin Top 50 pixel

Tabel 2. Margin One Side

unuk memberikan margin pada sisi lain cukup ubah mgt sesuai dengan label label berikut :

  • mgt untuk margin Top / Atas
  • mgb untuk margin Bottom / Bawah
  • mgl untuk margin Left / Kiri
  • mgr untuk margin Right / Kanan

Padding

Padding adalah jarak antara konten yang ada dalam atribut dengan koonten yang ada dalam atribut tersebut.

  • Padding All Side / Padding Semua Sisi

No Class Fungsi

1 pd-1 Memberikan Padding 1 pixel
2 pd-2 Memberikan Padding 2 pixel
3 pd-3 Memberikan Padding 3 pixel
4 pd-4 Memberikan Padding 4 pixel
5 pd-5 Memberikan Padding 5 pixel
n pd-n Memberikan Padding N pixel
50 pd-50 Memberikan Padding 50 pixel

Tabel 3. Padding All Side

  • Padding One Side / Padding Satu Sisi

Sama seperti border, Padding juga punya 4 sisi yang bisa kita atur ukuran Paddingnya masing masing yaitu Padding Bottom (Bawah), Padding Top (Atas), Padding Left (kiri) dan Padding Right (kanan).

No Class Fungsi
@ pdt-a Memberikan Padding Top Auto
@ pdt-0 Memberikan Padding Top 0 pixel
1 pdt-1 Memberikan Padding Top 1 pixel
2 pdt-2 Memberikan Padding Top 2 pixel
3 pdt-3 Memberikan Padding Top 3 pixel
4 pdt-4 Memberikan Padding Top 4 pixel
5 pdt-5 Memberikan Padding Top 5 pixel
n pdt-n Memberikan Padding Top N pixel
50 pdt-50 Memberikan Padding Top 50 pixel

Tabel 4. Padding One Side

untuk memberikan Padding pada sisi lain cukup ubah pdt sesuai dengan label label berikut :

  • pdt untuk Padding Top / Atas
  • pdb untuk Padding Bottom / Bawah
  • pdl untuk Padding Left / Kiri
  • pdr untuk Padding Right / Kanan

Display

Display

Display adalah bagaimana sebuah atribut ingin di tempatkan, Berjajar Horizontal, vertikal atau berada di depan atrbiut lain dan sejenisnya. Berikut kode yang bisa anda gunakan untuk memanggil fungsi display.

No Class Fungsi
1 none Memberikan Display none
2 inline Memberikan Display inline
3 inblock Memberikan Display inline block
4 inflex Memberikan Display inline flex
5 ingrid Memberikan Display inline grid
6 flex Memberikan Display felx
7 block Memberikan Display block
8 grid Memberikan Display grid

Tabel 1. Display

Position and Floats

Position (Posisi) dalam CSS mengontrol tata letak elemen di dalam halaman web. Properti ini memungkinkan Anda menentukan bagaimana elemen-elemen HTML ditempatkan relatif terhadap elemen lain atau terhadap viewport (area tampilan di layar).

No Class Fungsi
1 pst-ab Memberikan Position Absolute
2 pst-rt Memberikan Position relative
3 pst-fx Memberikan Position fixed
4 pst-st Memberikan Position static
5 pst-sk Memberikan Position sticky

Tabel 2. Position

Float adalah properti CSS yang digunakan untuk menggeser elemen ke samping dan memungkinkan elemen lain mengisi ruang di sekitarnya. Ini awalnya diciptakan untuk mengatur posisi gambar dalam teks, tetapi kemudian digunakan untuk membuat tata letak kolom yang kompleks.

No Class Fungsi
1 fl-l Memberikan Float Left
2 fl-r Memberikan Float Right
3 fl-c Memberikan Float Center

Tabel 3. Float

Overflow

Overflow adalah bagaimana sebuah atribut ingin di tempatkan, Berjajar Horizontal, vertikal atau berada di depan atrbiut lain dan sejenisnya. Berikut kode yang bisa anda gunakan untuk memanggil fungsi display.

No Class Fungsi
1 ovf-a Memberikan Overflow Auto
2 ovf-sc Memberikan Overflow Scroll
3 ovf-hn Memberikan Overflow Hidden
4 ovf-vb Memberikan Overflow Visible
5 ovf-ih Memberikan Overflow Inherit
6 ovf-it Memberikan Overflow initial

Tabel 4. Overflow

Jika anda ingin memberikan overflow pada posisi Y atau X anda bisa langsung saja tambahkan Y atau X seperti contoh berikut :
ovt-y-hd

Color

Font Color

Font Color adalah pilihan warna yang bisa anda gunakan untuk font/huruf website anda. Kode ini juga bisa digunakan untuk mengubah warna icon Font Awesome.

No Class Fungsi
1 c-white Memberikan warna Font Putih
2 c-black Memberikan warna Font Hitam
3 c-red Memberikan warna Font Merah
4 c-blue 1 -> c-blue-4 Memberikan warna Font Biru 1 -> Biru 4

Background Color

Background Color adalah pilihan warna yang bisa anda gunakan untuk baclground elemen pada website anda.

No Class Fungsi

4 bc-blue-1-t1 - bc-blue-3-t9 Memberikan warna Biru 3 dengan Transparansi 0.1 -> 0.9 pada background elemen
5 bc-blue-1-t1 - bc-blue-4-t9 Memberikan warna Biru 4 dengan Transparansi 0.1 -> 0.9 pada background elemen
6 bc-red-1 Memberikan warna Merah pada background elemen
7 bc-red-1-t1 - bc-red-1-t9 Memberikan warna Biru 1 -> Biru 4 pada background elemen
